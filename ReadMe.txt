JavaScript is an asynchronous programming language. This project helps to gain proficiency in asynchronous 

concepts. Buildig freeCodeCamp forum leaderboard, this project cover the Fetch API, promises, Async/Await, 

and the try..catch statement.

This section explains how to fetch leaderboard data from the FreeCodeCamp forum using JavaScript's async/await 

syntax to make asynchronous requests to the API.
